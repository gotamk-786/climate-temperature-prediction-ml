# Selected Proposal Papers

This folder contains the final set of topic-related papers kept for the TBW research proposal and the Data Science paper.

## Why These Papers Were Kept

These papers were selected because they are directly useful for the required related-work section. They focus on at least one of the following:
- temperature prediction
- climate change modeling
- machine learning or deep learning for climate variables
- South Asia or regional climate relevance
- comparative analysis of forecasting methods
