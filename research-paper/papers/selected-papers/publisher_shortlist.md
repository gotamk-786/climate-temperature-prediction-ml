# Publisher-Wise Shortlist for TBW Proposal

This file keeps a cleaner shortlist of papers that match the project topic:
`climate change / temperature prediction / machine learning`.

Note:
- `ScienceDirect` is Elsevier's platform, so some Elsevier papers also appear under ScienceDirect links.
- Only valid local PDFs are kept in the folder. Where direct PDF download was blocked, the article link is kept instead.

## MDPI

1. **Air Temperature Forecasting Using Machine Learning Techniques: A Review (2020)**
   - Fit for proposal: strong survey paper for the related-work section
   - Use for: methods overview, gaps, model families, forecast horizons
   - Local PDF: `publisher-papers/MDPI_Air_Temperature_Forecasting_Using_Machine_Learning_Techniques_A_Review_2020.pdf`
   - Article: https://www.mdpi.com/1996-1073/13/16/4215

2. **MSLKSTNet: Multi-Scale Large Kernel Spatiotemporal Prediction Neural Network for Air Temperature Prediction (2024)**
   - Fit for proposal: recent deep learning temperature prediction paper
   - Use for: modern architecture and performance discussion
   - Article: https://www.mdpi.com/2073-4433/15/9/1114

3. **A Review of Neural Networks for Air Temperature Forecasting (2021)**
   - Fit for proposal: useful secondary review for air-temperature forecasting literature
   - Use for: comparing ANN, RNN, LSTM, and short-term forecasting methods
   - Article: https://www.mdpi.com/2073-4441/13/9/1294

## Springer

1. **A Machine Learning-Based Model for Predicting Temperature Under the Effects of Climate Change**
   - Fit for proposal: directly aligned with the exact topic wording
   - Use for: linking machine learning with climate-change-driven temperature variation
   - Local PDF: `978-3-031-22456-0_4.pdf`

2. **Predicting temperature variability in major Indian cities using Random Forest Regression (RFR) Model (2025)**
   - Fit for proposal: very relevant regional paper using Random Forest on city temperature
   - Use for: comparing regression models, features, metrics, and regional context
   - Article: https://link.springer.com/article/10.1007/s12040-024-02501-1

## Elsevier / ScienceDirect

1. **Machine learning applications on air temperature prediction in the urban canopy layer: A critical review of 2011-2022 (2023)**
   - Fit for proposal: one of the best review papers for this topic
   - Use for: limitations, common models, gaps in evaluation and metadata reporting
   - Article: https://www.sciencedirect.com/science/article/pii/S2212095523000937

2. **Accurate long-term air temperature prediction with Machine Learning models and data reduction techniques (2023)**
   - Fit for proposal: direct air-temperature prediction study with ML and DL comparison
   - Use for: long-term forecasting, feature selection, and comparative modeling
   - Article: https://www.sciencedirect.com/science/article/pii/S1568494623001369

3. **A machine learning approach for the efficient estimation of ground-level air temperature in urban areas (2025)**
   - Fit for proposal: useful for urban climate and adaptation framing
   - Use for: climate resilience angle and high-accuracy estimation
   - Article: https://www.sciencedirect.com/science/article/pii/S2212095525001312

4. **Machine learning framework for high-resolution air temperature downscaling using LiDAR-derived urban morphological features (2024)**
   - Fit for proposal: useful if you want a stronger methodology comparison subsection
   - Use for: spatial downscaling and urban microclimate modeling
   - Article: https://www.sciencedirect.com/science/article/pii/S2212095524002992

## Taylor & Francis

1. **Prediction of hourly air temperature based on CNN-LSTM (2022)**
   - Fit for proposal: direct deep learning temperature prediction paper
   - Use for: CNN-LSTM baseline discussion and comparative methods
   - Article: https://www.tandfonline.com/doi/full/10.1080/19475705.2022.2102942

2. **Spatial Simulation and Prediction of Air Temperature Based on CNN-LSTM (2023)**
   - Fit for proposal: useful for spatial temperature prediction literature
   - Use for: temperature simulation plus deep learning methodology
   - Article: https://www.tandfonline.com/doi/full/10.1080/08839514.2023.2166235

3. **Artificial intelligence and remote sensing for spatial prediction of daily air temperature: case study of Souss watershed of Morocco (2021)**
   - Fit for proposal: relevant for regional daily air temperature prediction
   - Use for: remote-sensing features and Random Forest comparison
   - Article: https://www.tandfonline.com/doi/full/10.1080/10095020.2021.2014765
